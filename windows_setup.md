
💻 4. تعليمات تشغيل النظام (Windows) 🛠️
لضمان تشغيل مشروع MS SmartScan على بيئة ويندوز بنجاح، يرجى اتباع الخطوات المرتبة أدناه:

🔹 أولاً: تجهيز البيئة البرمجية
خادم الويب: تثبيت برنامج XAMPP وتفعيل خدمات Apache و MySQL من لوحة التحكم.

لغة البرمجة: تثبيت Python 3.9 أو إصدار أحدث (مع تفعيل خيار "Add Python to PATH").

المكتبات التقنية: افتح موجه الأوامر (CMD) ونفذ الأمر التالي لتثبيت محرك الذكاء الاصطناعي:

Bash
pip install ultralytics opencv-python pillow
🔹 ثانياً: إعداد الملفات وقاعدة البيانات
نقل الأكواد: فك ضغط ملف MS.smartScan.zip وانقله إلى المسار التالي:
C:/xampp/htdocs/

إنشاء القاعدة: توجه إلى phpMyAdmin وأنشئ قاعدة بيانات جديدة.

الاستيراد: اختر "Import" وارفع ملف database.sql المرفق لبناء الجداول والبيانات.

🔹 ثالثاً: تشغيل النظام
افتح المتصفح وادخل على الرابط التالي لبدء الفحص:
http://localhost/MS.smartScan/index.php

📂 5. هيكلة المستودع (Repository Structure)
📦 MS.smartScan.zip: يضم كافة الملفات المصدرية (PHP, Python, CSS) المهيأة للعمل.

🗄️ database.sql: الملف البرمجي لتوليد قاعدة بيانات المرضى والسجلات.

🧠 best.pt: ملف الأوزان الخاص بنموذج YOLO11m الذي تم تدريبه للتعرف على بؤر التصلب.
