# نظام MS SmartScan 🧠🔍
**نظام ذكي متكامل للكشف التلقائي عن بؤر التصلب المتعدد (MS) باستخدام الذكاء الاصطناعي.**

## 1. نبذة عن المشروع
مشروع **MS SmartScan** هو نظام طبي تقني يهدف إلى مساعدة الأطباء في تحليل صور الرنين المغناطيسي (MRI) للكشف عن مرض التصلب المتعدد. يجمع المشروع بين قوة الحوسبة الفيزيائية باستخدام جهاز **Raspberry Pi 5** ودقة نماذج التعلم العميق (Deep Learning) باستخدام نموذج **YOLO11m**.

يقوم النظام بالتقاط الصور وتحليلها فورياً لتحديد ما إذا كانت الحالة "مصابة" (Infected) أو "سليمة" (Healthy) مع تحديد دقة التنبؤ.

## 2. التقنيات المستخدمة (Technical Stack)
* **الذكاء الاصطناعي:** نموذج YOLO11m (النسخة المتوسطة لضمان دقة عالية في اكتشاف البؤر الصغيرة).
* **العتاد (Hardware):** جهاز Raspberry Pi 5 مع كاميرا (Camera Module V2)و (Display): شاشة مدمجة لعرض واجهة المستخدم والتفاعل مع النظام مباشرة.
* **لغات البرمجة:** PHP (للواجهة وإدارة السيرفر)، Python 3.11 (لمحرك الذكاء الاصطناعي).
* **قاعدة البيانات:** MySQL عبر بيئة XAMPP/MariaDB.
* **الواجهات:** HTML5, CSS, JavaScript.

## 3. مميزات النظام
* **دعم منصات متعددة:** يتضمن المشروع نسختين؛ واحدة مخصصة للرازبيري باي (Linux) وأخرى معدلة بالكامل لبيئة الويندوز (Windows).
* **التحليل الفوري:** معالجة الصورة وظهور النتيجة في ثوانٍ معدودة.
* **إدارة سجلات المرضى:** نظام كامل لتسجيل بيانات المرضى والرجوع لسجل الفحوصات السابقة.
* **واجهة مستخدم تفاعلية:** تصميم مريح وسهل الاستخدام يدعم اللغة العربية.
