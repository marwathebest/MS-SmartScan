# MS-SmartScan
نظام ذكي متكامل لتحليل الصور والكشف التلقائي عن بؤر التصلب المتعدد (MS)
Note for Supervisor:
To run the project on Windows:

Download and extract MS.smartScan.zip into C:/xampp/htdocs/.

Import database.sql into your local phpMyAdmin.

Install requirements: pip install ultralytics opencv-python.
