# نظام MS SmartScan 🧠🔍
**نظام ذكي متكامل للكشف التلقائي عن بؤر التصلب المتعدد (MS) باستخدام الذكاء الاصطناعي.**

## 1. نبذة عن المشروع
مشروع **MS SmartScan** هو نظام طبي تقني متكامل يهدف إلى مساعدة الأطباء في تحليل صور الرنين المغناطيسي (MRI) للكشف عن مرض التصلب المتعدد. يتميز المشروع بكونه "جهازاً مستقلاً" يجمع بين قوة الحوسبة الفيزيائية ودقة نماذج التعلم العميق (Deep Learning).

يقوم النظام بالتقاط الصور عبر الكاميرا المدمجة، ومعالجتها فورياً باستخدام نموذج ذكاء اصطناعي مدرب، ثم عرض النتائج مباشرة على الشاشة الخاصة بالنظام.

## 2. العتاد المستخدم (Hardware) 🛠️
تم اختيار القطع بعناية لضمان أداء عالي وسرعة في معالجة البيانات:
* **جهاز الحوسبة:** Raspberry Pi 5 (أحدث إصدار لضمان سرعة معالجة نموذج YOLO).
* **وحدة العرض (Display):** شاشة مدمجة لعرض واجهة المستخدم والتفاعل مع النظام مباشرة.
* **وحدة الكاميرا:** Raspberry Pi Camera Module V2 لالتقاط صور الأشعة بدقة عالية.
* **نظام التبريد:** مروحة تبريد نشطة لضمان استقرار المعالج أثناء تشغيل عمليات الذكاء الاصطناعي.
* **الطاقة:** مزود طاقة رسمي بقوة 5V 5A.

## 3. التقنيات البرمجية (Software Stack) 💻
* **الذكاء الاصطناعي:** نموذج **YOLO11m** (النسخة المتوسطة) الذي يوفر توازناً مثالياً بين الدقة والسرعة.
* **لغات البرمجة:** PHP (لإدارة السيرفر والواجهات)، Python 3.11 (لمحرك التحليل الذكي).
* **قاعدة البيانات:** MySQL (عبر بيئة XAMPP/MariaDB) لحفظ سجلات المرضى.
* **الواجهة الرسومية:** تصميم عصري باستخدام HTML5, CSS3 وخط القاهرة (Cairo Font).

## 4. مميزات النظام
* **تحليل فوري:** تظهر نتيجة الفحص (مصاب/سليم) مع نسبة الدقة في ثوانٍ معدودة.
* **نسخة الويندوز:** يتوفر ملف مضغوط جاهز للتشغيل على أجهزة الكمبيوتر العادية لسهولة العرض والمناقشة.
* **إدارة شاملة:** إمكانية إضافة مرضى جدد، واستعراض سجل الفحوصات السابقة لكل مريض.
* **استقلالية النظام:** بفضل الشاشة المدمجة، يعمل النظام كأداة طبية محمولة لا تحتاج لملحقات خارجية.
